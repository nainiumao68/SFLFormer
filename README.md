# SFLFormer
It's a spatial-frequency layered transformer frameworkc for RGB-T dual-modalities semantic segmentation task. Our network has achieved advanced or SOTA performance on six datasets, including MFNet, PST900, MSRS, FMB, CART, and SUSDataset. The relevant code of the network, including complete model files, training files, and testing files, will be open-sourced subsequently.

这是一个面向 RGB-T 双模态语义分割任务的空域-频域分层Transformer框架。我们的网络在MFNet、PST900、MSRS、FMB、CART和SUSDataset六个数据集上取得了先进性能或当前SOTA效果。该网络的相关代码（含完整的模型文件、训练文件及测试文件）后续将开源。
